El material

Hardware:

10 ordenadores portátiles HP, con los siguientes requisitos (incluyendo 5 ordenadores de repuesto):

  • Sistema operativo: Windows 11 Home en modo S
  • Procesador: Intel N-series N100
  • Memoria: RAM 4 GB RAM
  • Disco duro: 128 GB UFS
  • Pantalla: 39,6 cm (15,6 pulg.), FHD (1920 x 1080)
  • Tarjeta Gráfica: Gráficos Intel UHD

1 sistema de almacenamiento NAS Synology para compartir archivos en red:

  • Procesador: Intel Celeron J4025 de 2 núcleos a 2,0 GHz, con turbo de hasta 2,9 GHz
  • Memoria: DDR4 de 2 GB integrada (ampliable hasta 6 GB)
  • Disco duro:2 x HDD/SSD SATA de 3,5" o 2,5"
  • LAN:2 x Gigabit (RJ-45)

También disponíamos de 10 móviles para el trabajo (incluyendo 5 móviles más de repuesto), el modelo era el Google Pixel 7A.

Software:

Todo el software utilizado es Open Source, como ya había sido mencionado más arriba.

La combinación de JAVA y Netbeans 8.2 resultó ser muy útil a la hora de programar una aplicación de esta complejidad.

Las bases de datos SQL resultaron mucho más cómodas de programar con Hibernate y JPA.

GitHub fue muy útil para compartir código con los compañeros de trabajo.

Odoo fue indispensable para saber que estaba haciendo cada uno de nostoros en cada momento.

Todoist nos sirvió para gestionar las tareas y no perdernos por el camino.

Necesitábamos contactar también a veces fuera del trabajo para pulir detalles.

Drive es una herramienta para compartir que no puede faltar en ningún proyecto.
