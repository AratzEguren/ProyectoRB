Objetivo del proyecto

La creación de esta aplicación de banco estaba basada en la simplicidad. El objetivo es crear una aplicación que una y simplifique las operaciones más usadas en el día a día de un banco.

Esta aplicación se diseñó con la intención de facilitar las varias operaciones que un banco ejecuta miles de veces durante sus días laborales. Todas las operaciones más relevantes como transferencias, creación de los varios servicios, ingresos, etc. tenían que poder ejecutarse rápidamente simplemente con la pulsación de unos botones.

Operaciones más importantes:

  • Cuentas Bancarias
  • Manejo de tarjetas de credito
  • Traspasos de dinero
  • Conceder préstamos

Objetivos Específicos

Aplicación intuitiva

Cualquier usuario que nunca ha visto la aplicación podrá entender su uso simplemente con la interfaz que se le presenta. Usamos una interfaz sencilla pero potente.

Fácil manejo

La aplicación tiene diferentes ventanas cuyo diseño se hizo teniendo en mente la facilidad y la simplicidad. Cada operación principal del banco se puede manejar sencillamente con los varios botonoes y menus dispuestos de manera clara en la interfaz. Unos simples clics es todo lo que se necesita para llevar a cabo las operaciones que nuestra aplicación proporciona

Segura

Los datos manejados en un banco son muy delicados. Debido a eso queremos asegurarnos que solo el personal autorizado tiene acceso. Para cumplir tal objetivo usamos encriptación para el manejo de la contraseña de los usuarios. No solo nos aseguramos que las contraseñas viajen lo minimo posible por seguridad, pero encima usamos una encriptacion avanzada para asegurarnos que nadie puede robarla durante el envío. Las contraseñas se guardan hasheadas de manera segura en nuestra base de datos. En caso de perdida, olvido o si simplemente por seguridad se requiere cambiar de contraseña también nos aseguramos de que el cambio se haga de manera rápida y segura.

Fácil navegación

Todas las operaciones que se pueden realizar a través de la aplicación están unidas por una interfaz y menus muy sencillos y claros.
