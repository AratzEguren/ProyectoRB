Descripción del Contexto

El objetivo principal

La creación de esta aplicación de banco estaba basada en la simplicidad. Se quería una aplicación que uniese las operaciones más usadas en un banco. El objetivo era simplificar y unificar las operaciones que el banco lleva a cabo diariamente.

Aplicación intuitiva

Queríamos que un usuario que nunca ha visto la aplicación supiese solo con verlo como funciona la aplicación. Usamos una estructura básica.

Facil manejo

La aplicación y sus diferentes ventanas tienen varias operaciones que se pueden llevar a cabo con unos clics a través del uso de botones.

Segura

Los datos manejados en un banco son muy delicados. Debido a eso usamos encriptación para el manejo de la contraseña de los usuarios. De forma estándar también guardamos las contraseñas hasheadas.

Facil navegacion

Una vez iniciada sesión en la aplicación la navegación a través de la aplicación queríamos que fuese sencilla. Creamos una barra de menú desde donde se puede acceder a todas las opciones que el programa puede ofrecer.

Soporte

La aplicación tiene su documento de ayuda donde se explica el uso de cada ventana así como las operaciones para las que se han diseñado cada ventana.
